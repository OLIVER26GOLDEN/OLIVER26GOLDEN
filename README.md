## 🤖 What I Actually Build

> I don't just write Python — I build **automated trading signal systems** that monitor financial markets, detect breakout/resistance patterns, and deliver real-time alerts directly to Telegram.

### How my bots work:

```
┌─────────────────────────────────────────────────────────────┐
│                    TRADING BOT PIPELINE                      │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  IQ Option API  ──►  Signal Detection  ──►  Telegram Alert  │
│                                                              │
│  • Connect to        • Breakout logic      • Asset name      │
│    IQ Option         • Resistance lvl      • Direction ↑↓    │
│  • Fetch OHLC        • Multi-asset         • Support level   │
│    candle data         via threading       • Amount          │
│  • Monitor OTC       • Configurable        • Martingale lvl  │
│    assets              asset list          • Timestamp       │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

**Key design decision:** bots send **signals only** — no auto-execution. The trader decides when to act.

### 🤖 [TeamBinari_Bot](https://github.com/OLIVER26GOLDEN/TeamBinari_Bot)

> Multi-asset trading signal bot connected to IQ Option. Monitors assets in parallel using Python `threading` and fires real-time alerts to a Telegram channel with full signal breakdown.

**Signal payload sent to Telegram:**
```

📈 Direction:      CALL ↑
🛡️ Support Level:  1.2345
💰 Amount:         $10
🔁 Martingale:     Level 1
🕐 Time:           14:32:05 UTC
```

**Stack:** `Python` `IQ Option API` `Telegram Bot API` `threading` `websockets`
